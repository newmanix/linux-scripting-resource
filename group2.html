Group 2: Loops in Shell Script

Body of a Loop - The single statement or a block of statements that can be repeated (a loop may not execute at all if the condition is false to start with). In Java the body of the loop is either the first statement following a while or for loop is the body of the loop or a block of statements enclosed in { and }.

For Loop - A loop that has a header with 3 optional parts: initialization, condition, and change. It does the initialization one time before the body of the loop executes, executes the body of the loop if the condition is true, and executes the change after the body of the loop executes before checking the condition again.

For Each Loop - Used to loop through all the elements of a collection (like a list or an array) and each time through the loop set a variable to the next item from the collectionInfinite Loop - A loop that never ends.

Loop - A way to repeat one or more statements in a program.Nested Loop - One loop inside of another.Out of Bounds error - A run-time error that occurs when you try to access past the end of a string or list in a loop.Trace Code - Writing down the values of the variables and how they change each time the body of the loop executes.While Loop - A loop that repeats while a Boolean expression is true.
