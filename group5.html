Group 5: ﻿Node Package Manager

A script written using the Node.js Package Manager to automate the creation and interaction of a Git repository. The script provides the function to start a repository, commit to a repository and view the status of a repository via commands groups that are easily executed using the following commands:

  • postgitAll: Command will alter user of a successful commitment and run a git status command to show a clean workspace
  • gitAll: Command will execute the git add . and git commit commands sequentiually. This will allow a user to add a change to the staging area and commit the change within one command.
  • start: Command will start a server to establish a repository within.
